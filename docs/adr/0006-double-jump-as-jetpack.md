# 0006 — 二段跳 = 喷气背包（物理化扩展，不算魔法）

## Context

二段跳在主流 2D 平台游戏里是事实标准，但物理上违反纯物理（空中无反作用质量）。原文档 §3.2 列为 v1.5 加入项，未指定哲学路径。

讨论中识别出两种实现路径：
- A. 承认魔法 —— 空中直接给冲量，代码标注 magic
- B. 物理化包装 —— 角色背"喷气背包"，向下喷气获得反作用力

## Decision

**采用 B：喷气背包模型**。二段跳 = 角色背包向下喷气体 → 反作用力推角色向上 → 等效一个向上冲量。

实现：
- `jetpack_charges` 内部计数器（不一定可见 UI）
- 起跳时 if charges > 0 → 给向上冲量，charges -= 1
- 落地（接地态切换为 true）→ charges 重置

## Rationale

- 跟 [ADR-0001 内在发动机派](0001-inner-engine-school.md) 一脉相承：第一段跳的反作用源是**地面**，二段跳的反作用源是**气体**，都是反作用力，本体论一致
- 不是魔法 —— 反物理动作有了物理叙事，无需 §2.5 意义上的"魔法开关"
- 跟项目愿景兼容：未来"喷气背包升级"、"过热冷却"、"燃料耗尽"等 setpiece 自然可玩

## 后续机制方法学

确立**通用方法学：遇到看似违反纯物理的玩法机制，先尝试物理化包装；包装不下的才考虑承认魔法。**

适用扩展（未在 v1）：
- Dash → 大喷气（一次大冲量）
- 墙跳 → 蹬墙反作用力（物理上真实可能）
- 钩爪 → 绳张力（Box2D 距离关节天然支持）
- 抓边 → 手指摩擦 + 关节张力 + 体力计
